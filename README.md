# into_all_scales
An instrument to transform musical patterns to all scales.

### Local setup
```shell
$ git clone git@github.com:Eira/into_all_scales.git
$ cd into_all_scales
$ python3.11 -m venv venv
$ source venv/bin/activate
$ pip install -U poetry
$ poetry install
```

### Local run tests
```shell
$ python -m pytest
```
